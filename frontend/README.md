## Getting Started

## Add this to your .env.local file:

```bash
NEXT_PUBLIC_API_URL=http://localhost:3000
```

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:4000](http://localhost:4000) with your browser to see the result.
